# Playwright LambdaTest Certification

This repository holds three Playwright test scripts for the LambdaTest Selenium Playground assignment.

## Test Scripts

1. tests/simpleFormDemo.js  
2. tests/dragDropSliders.js  
3. tests/inputFormSubmit.js  

## Setup

1. Clone the repository.  
2. In the project root run:  
   npm install  
   npx playwright install  

## Run Tests

- Run all tests:  
  npm test  
- Run a single test:  
  npm run test:simple  
  npm run test:slider  
  npm run test:form
